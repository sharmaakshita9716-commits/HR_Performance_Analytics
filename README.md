Project overview:

This project analyzes the IBM HR Analytics dataset to understand why employees leave and builds a predictive model to flag at-risk talent.

Key insights:

The primary driver of attrition is overtime.
Employees earning less than $5k show highest turnover.
Accuracy:84.35%(Baseline Random Forest)
SMOTE implementation.Improved attrition recall by 2.9x (from 13%to 38%)

Tech Stack:
Python(Pandas,Seaborn,Matplotlib)
Scikit-learn(Random Forest,SMOTE)
Google colab
